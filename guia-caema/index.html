Agrupamento e Desagrupamento

Documentos Necessários

Tanto para agrupamentos ou desagrupamentos, existe a necessidade de apresentação de Documentos pela parte solicitante.

- Ofício com a solicitação (tanto para agrupamentos quanto desagrupamentos); Contrato de locação (para agrupamentos); Destrato e/ou termo de entrega de chaves (para desagrupamentos);

OBS.: O que define se uma matrícula está Agrupada é a definição de um Responsável para o Imóvel e se a opção de envio de conta estiver definido como: Enviar para o cliente responsável - Não emite conta.
Passo à passo para Agrupamentos

Existe duas formas para podermos agrupar uma matrícula:

01 - Usando a funionalidade de manter imóvel em: GSAN -> Cadastro -> Imovel -> Manter Imovel, para aqueles imóveis que não possuem débitos anteriores e não existe a necessidade de alterar o vínculo das contas;

02 - Usando a funionalidade de alterar o vínculo da conta em: GSAN -> Faturamento -> Conta -> Alterar Vinculos de Clientes com Imovel e Contas , para os imóveis que possuem débitos anteriores e precisamos definir a responsabilidade de cada cliente para cada período da dívida;
